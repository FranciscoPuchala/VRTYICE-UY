vDTM - DownTime Manager

Gestión de Tiempos Muertos

vDTM ayuda a identificar y cuantificar las pérdidas por tiempos muertos. Proporciona indicadores que se basan en formulaciones estándares de la industria.

Sistema Completamente Integrado

Permite modelar la planta con sus áreas, líneas y unidades de producción de manera intuitiva y eficiente.

Recolección en Tiempo Real

Captura información de piso de planta en tiempo real y la registra en un servidor de información centralizado.

Visualización Clara

Proporciona representación numérica y gráfica clara, tanto en tiempo real como sobre cualquier período histórico.

Fácilmente Configurable

Configurable por el usuario sin requerir conocimientos especiales de programación, maximizando la flexibilidad operativa.

Optimización máxima de instalaciones productivas
Retorno de inversión inmediato

vHWS - Historian Web Services

Servicios Web para Historian

Ofrece todo el potencial de consultas a uno o más servidores Proficy Historian vía Web Services. Las mismas incluyen todas las variantes de pedido de datos, tal como se hace con el Excel Add-In.

Conectividad Múltiple

Acceso simultáneo a múltiples servidores Proficy Historian desde cualquier aplicación compatible con Web Services.

Variantes de Consulta Completas

Incluye todas las variantes de pedido de datos disponibles, replicando la funcionalidad del Excel Add-In.

Tecnología de Web Services

Utiliza protocolos y estándares web modernos para intercambio eficiente de datos entre aplicaciones.

Integración Simplificada

Facilita la integración con sistemas de terceros y aplicaciones corporativas de manera estandarizada.

Acceso flexible a datos históricos
Consultas de alto rendimiento

vMail - iFIX Mail Services

Notificaciones Inteligentes

vMail brinda una solución flexible que cubre todas las necesidades de comunicaciones entre sistemas de supervisión y control basados en el software Proficy HMI / SCADA iFIX con el correo electrónico.

Configuración Flexible

Envío de mensajes conteniendo alarmas, eventos, valores en tiempo real o indicadores claves del proceso de forma personalizable.

Gestión de Destinatarios

Configuración individual o por grupos de destinatarios, con total flexibilidad en la distribución de notificaciones.

Envío Condicional Programable

Funcionalidad de envío por horario, día de la semana o fecha especial, modificando destinatarios según necesidad para cada evento independiente.

Consultas Seguras

Usuarios certificados pueden enviar mails codificados solicitando datos del proceso. vMail responde automáticamente con la información solicitada.

Adaptabilidad Total

Capacidad de adaptación a distintos escenarios y requisitos particulares, ajustándose a las necesidades específicas de cada operación.

Contacto continuo con el proceso
Comunicación segura y confiable
