Proficy Historian

Inicio / Productos / Historiador / Historian

El Corazón de sus Datos Industriales

La excelencia operativa requiere que se recolecten y se optimicen grandes cantidades de datos a través de los procesos de producción, para luego permitir una clara visión de los mismos.

Proficy Historian es el corazón recolector y distribuidor de todos los datos de entrada y salida de los procesos. Ofrece ventajas claves que incluyen velocidades sin precedentes de recopilación de datos, compatibilidad de toma de datos de muy diversas fuentes, mejor compresión, redundancia robusta, seguridad y auditoría y distribución de los datos a velocidades asombrosas.

Características Principales

Alta Velocidad

Velocidades sin precedentes de recopilación de datos y distribución a clientes con velocidades asombrosas.

Múltiples Fuentes

Compatibilidad con servidores SCADA, OPC, bases de datos relacionales, archivos CSV/XML y aplicaciones personalizadas.

Seguridad y Redundancia

Mejor compresión, redundancia robusta, seguridad y auditoría para garantizar la integridad de sus datos.

Integración Abierta

Integración fácil con aplicaciones corporativas mediante tecnologías abiertas como OLE DB y consultas SQL estándar.

Beneficios para su Operación

Como resultado de estas capacidades, Proficy Historian puede aprovecharse para mejorar la visibilidad del proceso productivo, acelerar la toma de decisiones, aumentar la productividad y reducir los costos.

Visibilidad del Proceso

Mejore la visibilidad de sus procesos productivos con datos precisos en tiempo real

Decisiones Aceleradas

Acelere la toma de decisiones con información integrada y accesible

Aumento de Productividad

Incremente la productividad aprovechando el análisis de datos históricos

Reducción de Costos

Reduzca costos operativos mediante la optimización basada en datos

Uniendo Islas de Información

Proficy Historian une las islas de información de automatización sin comprometer la resolución de los datos y proporciona una ventana a las operaciones de fabricación. Los datos recolectados, adecuadamente combinados y procesados, sirven como el vínculo vital entre las operaciones de la planta y los sistemas de negocios, ofreciendo una visión integrada de sus operaciones con información precisa en tiempo real.

Consultas SQL Estándar

Consulte datos, alarmas y eventos mediante consultas SQL estándares

Tecnologías Abiertas

Integración mediante OLE DB con aplicaciones y sistemas corporativos

Fuentes de Datos Soportadas

Tiene tecnologías incorporadas que permiten recolectar datos desde muy diversas fuentes:

Servidores SCADA

Conexión directa con sistemas SCADA para recopilación de datos en tiempo real

Servidores OPC

Compatible con cualquier servidor OPC para máxima flexibilidad

Bases de Datos

Consultas SQL configurables desde bases de datos relacionales

Archivos CSV/XML

Importación de datos desde archivos CSV y XML

Aplicaciones Personalizadas

APIs para integración con aplicaciones desarrolladas a medida

Rendimiento Excepcional

Proficy Historian puede recopilar grandes volúmenes de datos de planta en tiempo real a velocidades increíblemente elevadas y es capaz de distribuirlas a clientes con velocidades también sorprendentes.

Esta capacidad de alto rendimiento asegura que sus operaciones cuenten con la información más actualizada en todo momento, permitiendo respuestas rápidas y decisiones informadas basadas en datos en tiempo real.
