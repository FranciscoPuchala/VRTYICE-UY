Curso Proficy Historian

El curso es una introducción al concepto de contenedor de datos de tiempo real.

El usuario podrá administrar, configurar la recolección de datos de diversos colectores, realizar consultas y crear reportes con el complemento Excel Add-In del Historian.

Está orientado a los usuarios de Historian, a ingenieros o integradores de automatización y a las personas vinculadas al área de tecnologías de información.

Requisitos

  • Conocimientos operativos de aplicaciones de control de procesos, sensores, autómatas y su conexión a PCs
  • Noción y manejo de sistemas operativos Windows 10 o posterior y su configuración
  • Conocimientos básicos de Microsoft EXCEL
  • Familiaridad con otros componentes del Software Proficy, especialmente con iFIX
  • Nociones de intercambio de datos a nivel de Tecnología de Información

Duración

El curso tiene una duración de 8 horas, dividido en 2 clases.
